# Vue 3 + TypeScript + Vite + Element-plus

> 这是一个极简的 PC-web 网页模板，开箱即用，它只包含了基础必备功能：axios封装、Element-plus按需引用。

## 环境要求

|node|
|----|
|`18.20.3`|
